자바스크립트 이벤트 정리

이벤트 사용 위치 설명
onmouseover / onmouseout 메인 페이지 - 카드 마우스를 올리면 강조, 벗어나면 원상복귀
onload 회원가입 페이지 생년월일 select 박스를 자동 생성
onchange 회원가입 페이지 생년월일 select 값 변경 시 나이 자동 계산
onclick 회원가입 페이지 - 가입 버튼 입력값 alert 출력 + 폼 제출 방지
onfocus / onblur 회원가입 페이지 - input 입력창 선택 시 강조, 벗어나면 원상복귀
onwheel 이벤트 목록 페이지 휠 스크롤 시 2칸씩 부드럽게 이동
