이벤트 목록

이벤트 사용 위치 설명
addEventListener 메인 페이지 - 휠 이벤트, 네비게이션 햄버거 버튼 이벤트등
자기소개 페이지 - 마우스 움직일때 이벤트등
브라우저에서 특정 이벤트가 발생했을 때 실행할 함수를 등록
mouseenter 자기소개 페이지 - 타이핑효과 마우스를 해당 객체에 올렸을 때 타이핑효과
mousemove 자기소개 페이지 - 마우스에 따라 객체 움직임 마우스 위치에 따라 이미지 움직임
onclick 회원가입 페이지 - 가입 버튼 입력값 alert 출력, 폼 제출
wheel 메인 페이지 - 휠 이벤트 휠로 섹션 간 스크롤 이동 (상하방향감지)
scroll 메인 페이지 - 섹션 별 분류 현재 화면 위치에 따라 네비게이션 및 섹션 활성화 갱신
